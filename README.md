# GESTURES-GAME

Gesture Controlled Snake Game

A real-time Snake Game controlled using hand gestures through a webcam.

This project uses:

OpenCV for camera input
Media Pipe for hand tracking
Pygame for game development

How It Works:

The webcam detects your hand using Media Pipe.
The index finger position determines snake direction.

Move your hand:

Left → Snake moves left
Right → Snake moves right
Up → Snake moves up
Down → Snake moves down

Features:
Real-time hand gesture detection
Smooth snake movement
Food collision detection
Game over detection
Dual window display (Game + Camera view)
